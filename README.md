# Notes App - Android

A simple and elegant notes application built with **Kotlin** and **Jetpack Compose** using Material Design 3.

## 📱 Features

- ✅ Create new notes with title and content
- ✅ View all notes in a scrollable list
- ✅ Delete notes with one tap
- ✅ Timestamp for each note
- ✅ Material Design 3 UI
- ✅ In-memory storage (notes reset on app restart)
- ✅ Empty state message

## 🛠️ Technologies Used

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Architecture:** MVVM (Model-View-ViewModel)
- **Minimum SDK:** API 24 (Android 7.0)
- **Target SDK:** API 35


## 🚀 How to Run

### Prerequisites
- Android Studio Hedgehog (2023.1.1) or later
- JDK 11 or higher
- Android SDK with API 35
